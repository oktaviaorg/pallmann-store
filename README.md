# Ponceur-Parquet.fr

Site web pour demandes de devis de ponçage et rénovation de parquet.

## Création d'articles

Pour créer de nouveaux articles de blog, utilisez les scripts de publication :

```bash
# Publier un article spécifique
npm run ts-node publish_article.ts

# Publier plusieurs articles
npm run ts-node publish_multiple_articles.ts
```

## Scripts disponibles

- `publish_article.ts` - Publier l'article "Parquet abîmé par locataire"
- `publish_escalier_colmar.ts` - Publier l'article "Rénovation escalier Colmar"
- `publish_multiple_articles.ts` - Publier plusieurs articles en une fois

## Développement

```bash
npm run dev
```

## Build

```bash
npm run build
```
